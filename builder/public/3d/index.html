Ukuran / Area Default

Kotak putus-putus = ukuran standar. Pakai tombol ini agar semua gedung seukuran & konsisten.

Katalog Model ⟳

Taruh .glb di v2/public/models/ lalu daftarkan di models.json. Klik untuk taruh di tengah, atau drag ke lantai.

Tembok

Klik titik demi titik. Enter/dobel-klik = selesai · Esc batal. “Lurus” mengunci sudut ke kelipatan 15° dan menampilkan panjang/sudut.

Pintu / Jendela

Klik pada sebuah tembok untuk melubanginya. Ambang bawah 0 = pintu; >0 (mis. 1) = jendela.

Tembok Terpilih

Titik kuning di sudut tembok bisa digeser (drag) untuk ubah bentuk.

Lubang (Pintu/Jendela)

Lantai (baru)

Klik 2 sudut untuk kotak lantai. Elevasi > 0 = platform tinggi; Ramp/Tangga naik dari 0 ke elevasi ke arah yang dipilih.

Lantai Terpilih

“Ke Depan” menaruh lantai ini di atas yang lain (mis. jalur hijau di atas beton).

Pin Device

Klik di lantai untuk menaruh titik status. Pin ber-IP menyala hijau/merah di dashboard.

Teks

Ketik isi lalu klik di lantai untuk menaruh. Pilih teks yang ada untuk mengubah isinya.

Model Terpilih

Pastikan file model juga ada di public/models/ sesuai Path, agar dashboard bisa memuatnya.
